Download

Several versions are availables depending on your needs: Eclipse package (recommanded for most usages), Virtual machine (for OS independant demos running in Virtual Box) or update site (in order to install GEMOC into an existing eclipse).
Requirements for Eclipse packages and update site: Java 8 and JavaFX . JavaFX is required for the Multidimensional timeline features, please make sure to install it or to use Oracle JRE.
Getting started: The studio comes with various example languages that you can import in your workspace: _File > New > Examples…_. — You can also install additional examples using the component discovery service: _Help > Install additional GEMOC components_. — You can also browse the [language gallery](http://gemoc.org/gallery/) for some more examples.
