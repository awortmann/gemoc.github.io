Update sites corresponding to additional GEMOC components
The GEMOC Component Discovery Catalog lists components that are not directly included in the Studio but that provides interresting new features of example languages. The catalog page also list the update sites if you wish to install their content using the classic Eclipse Install new software... feature instead of the dedicated Install Additional GEMOC Component feature.
